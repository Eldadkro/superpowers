# Pi Superpowers Manual Verification

Run these checks in an interactive Pi session from this worktree:

1. Start a fresh Pi session in this repository.
2. Confirm Superpowers skills are absent in normal mode.
3. Run `/super` and confirm `SUPER ON` appears and Superpowers skills become available after reload.
4. Run `/normal` and confirm the indicator clears and Superpowers skills disappear after reload.
5. Run `/super fix this bug` and confirm the task is forwarded once after reload.
6. Run `/reload` and confirm the active mode persists.
7. Run `/new` and confirm the new session starts in normal mode.
